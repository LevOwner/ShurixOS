# ShurixOS

**ShurixOS** — это лёгкий 32-битный гипервизор, который позволяет запускать любые операционные системы (Windows, Linux, macOS) в контейнерах без эмуляции железа.

---

## ⚡ Возможности

- ✅ **Свои форматы контейнеров** — `.cosa` и `.acos`
- ✅ **Свои файловые системы** — CFS (настройки) и HSFS (образы)
- ✅ **Свои протоколы** — KAFOS, KCBUC, KCBUSP, KCBUOTM, KCBUMSA
- ✅ **BIOS-подобный интерфейс** — синий фон, навигация стрелками
- ✅ **Экономия места** — контейнеры занимают до 90% меньше места
- ✅ **Быстрое переключение между ОС** — за 5 секунд

---

## 🛠️ Как собрать

### Требования
- **QEMU** (эмулятор)
- **GCC + NASM** (компиляторы)
- **i686-elf-gcc** (кросс-компилятор)
